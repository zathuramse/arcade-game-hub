# 霓虹蛇域 Neon Snake Arena

一個單頁 Canvas 貪食蛇遊戲。核心目標是做出比前一個太空射擊專案更完整的完成度：更多狀態系統、更多畫面特效、可升級流程、手機操作與可直接本機執行。

## 執行

直接開啟 `index.html` 即可遊玩。若要用本機伺服器：

```powershell
python -m http.server 4174
```

然後開啟 `http://localhost:4174`。

## 操作

- 方向鍵 / WASD：轉向
- Space / Shift：加速
- F：專注慢速
- E：脈衝清場
- Q：相位穿越，需先升級解鎖
- P：暫停
- R：重新開始
- 手機可使用畫面下方方向鍵與能力按鈕

## 內容

- 霓虹光軌蛇體、粒子爆裂、衝擊波、掃描線、動態星場與畫面震動
- 分區段進度、連擊、結晶貨幣與升級艙
- 脈衝雷、閃電門、傳送裂縫、每 4 區段守門者事件
- 引擎、磁吸、護盾、相位四條升級線
- Boost、Focus、Pulse、Phase 四種主動能力與冷卻顯示

## 檔案

- `index.html`：頁面結構與 UI
- `styles.css`：響應式介面、HUD、控制列與面板
- `game.js`：遊戲狀態、物理節奏、生成、碰撞、渲染與輸入
- `ITERATION_LOG.md`：製作與自我迭代紀錄
