# Iteration Log

## v1 - 可玩核心

- 建立獨立專案 `neon-snake-arena`。
- 完成 Canvas 貪食蛇核心循環：方向緩衝、步進移動、吃核心、成長、自身與邊界碰撞。
- 加入 HUD、開始畫面、區段突破畫面、結束畫面。

## v2 - 視覺升級

- 加入霓虹蛇體漸層、頭部眼睛、星場、掃描線、格線、暗角。
- 加入粒子、浮動分數、衝擊波、畫面震動與閃光。
- 食物與道具使用旋轉發光符號，而不是普通方塊。

## v3 - 系統深度

- 加入區段進度、目標核心、連擊、結晶貨幣。
- 加入脈衝雷、閃電門、傳送裂縫與每 4 區段的守門者事件。
- 加入 Boost、Focus、Pulse、Phase 四種能力，並顯示冷卻。

## v4 - 長線流程

- 加入區段突破後的升級艙。
- 升級線包括引擎、磁吸、護盾、相位。
- 護盾可抵消碰撞，相位可穿牆與穿越自身。
- 磁吸會把核心與道具拉近，讓後期節奏更流暢。

## v5 - 可用性

- 加入響應式版面與手機方向鍵。
- 保持無外部依賴，直接開 `index.html` 或使用簡單本機伺服器即可玩。
- 補上 README 與操作說明。

## v6 - QA 後調整

- 用 Headless Chrome 檢查桌面與手機寬度。
- 修正快速連續輸入造成開局非預期自撞的問題，方向緩衝改為單步。
- 補上開場預覽蛇體，避免開始前 HUD 顯示空長度。
