# 星光跑者 Starlight Runner

原創橫向卷軸平台冒險遊戲，靈感來自經典跳躍冒險，但不使用既有商標角色或素材。

## 執行

直接開 `index.html` 可玩。若要使用本機伺服器：

```powershell
python -m http.server 4190
```

然後開啟 `http://127.0.0.1:4190`。

## 操作

- A / D 或方向鍵：左右移動
- Space / W / 上：跳躍
- Shift：衝刺
- Enter：開始 / 繼續
- R：重新開始
- 手機可用畫面下方觸控按鈕
- 開始遊戲後會啟動合成背景音樂與音效

## 內容

- 16:9 舞台會依視窗高度縮放，避免看不到主角與環境全貌
- 橫向卷軸鏡頭、平台碰撞、重力、加速度、摩擦、衝刺
- 跳躍緩衝與土狼時間，讓跳躍手感更順
- 調高跑速、跳高與衝刺跳距，普通跳可撞問號磚，衝刺跳可跨越常見缺口
- 金幣、星磚、問號磚、磚塊頂撞與道具生成
- 敵人巡邏、跳躍型敵人、踩踏擊倒、受傷與復活
- 檢查點、終點門、兩段關卡流程
- 道具：護盾、跳靴、破磚火花
- 粒子、浮動分數、跑步塵土、背景雲層與多層山景
- Web Audio 合成背景音樂、跳躍、金幣、踩敵、受傷、道具、通關音效
- 響應式介面與手機操作

## 檔案

- `index.html`：頁面與 HUD
- `styles.css`：響應式 UI 與視覺樣式
- `game.js`：平台物理、關卡、敵人、道具、渲染
- `ITERATION_LOG.md`：製作與 QA 迭代紀錄
- `VERSION_300_UPGRADE_LOG.md`：300 版升級紀錄
